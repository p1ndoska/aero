# Функция "Версия для слабовидящих"

## Описание

Реализована полная функциональность "Версия для слабовидящих" согласно требованиям ГОСТ 52872-2019 для обеспечения доступности сайта для людей с нарушениями зрения.

## Расположение

Кнопка "Версия для слабовидящих" находится в левом сайдбаре сайта, в разделе с поиском и кнопками входа.

## Функциональность

### 1. Переключатель режима доступности
- Включает/выключает специальные настройки для слабовидящих
- Сохраняется в localStorage

### 2. Настройки текста
- **Размер шрифта**: от 100% до 300% (рекомендуется 200%)
- **Межстрочный интервал**: от 1.0x до 2.0x (рекомендуется 1.5x)
- **Межбуквенный интервал**: от 0px до 2px

### 3. Цвета и контрастность
- **Контрастность**: Обычная / Высокая (7:1 согласно ГОСТ)
- **Цветовая схема**: Обычная / Темная / Инвертированная

### 4. Дополнительные настройки
- **Уменьшить анимации**: для пользователей с вестибулярными расстройствами
- **Подсветка фокуса**: четкое выделение активного элемента

## Соответствие ГОСТ 52872-2019

✅ **Контрастность**: коэффициент контраста не менее 7:1  
✅ **Увеличение шрифта**: до 200% без потери функциональности  
✅ **Межстрочный интервал**: 1.5x по умолчанию  
✅ **Навигация с клавиатуры**: поддержка Tab и Enter  
✅ **Поддержка скрин-ридеров**: семантическая разметка  
✅ **Упрощенный дизайн**: без лишних элементов  
✅ **Цветовое выделение**: с дополнительными визуальными отличиями  

## Технические особенности

### CSS классы
- `.accessibility-mode` - основной класс режима доступности
- `.high-contrast` - высокая контрастность
- `.color-scheme-dark` - темная схема
- `.color-scheme-inverted` - инвертированная схема
- `.reduce-motion` - уменьшение анимаций
- `.show-focus` - подсветка фокуса

### CSS переменные
- `--accessibility-font-size` - размер шрифта
- `--accessibility-line-height` - межстрочный интервал
- `--accessibility-letter-spacing` - межбуквенный интервал

### Сохранение настроек
Все настройки сохраняются в localStorage и восстанавливаются при следующем посещении сайта.

## Использование

1. Нажмите кнопку "Версия для слабовидящих" в сайдбаре
2. Включите переключатель "Включить версию для слабовидящих"
3. Настройте параметры по своему усмотрению:
   - Размер шрифта
   - Межстрочный интервал
   - Контрастность
   - Цветовую схему
4. Нажмите "Применить"
5. Настройки будут применены ко всему сайту

## Сброс настроек

Для возврата к стандартным настройкам нажмите кнопку "Сбросить" в диалоге настроек.

## Поддержка браузеров

Функция работает во всех современных браузерах:
- Chrome 80+
- Firefox 75+
- Safari 13+
- Edge 80+
