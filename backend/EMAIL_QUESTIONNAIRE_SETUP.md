# Настройка отправки анкеты потребителя по email

## Настройка Gmail

1. Включите двухфакторную аутентификацию в вашем Google аккаунте
2. Создайте пароль приложения:
   - Перейдите в настройки Google аккаунта
   - Безопасность → Пароли приложений
   - Создайте новый пароль для приложения
   - Используйте этот пароль в переменной EMAIL_PASS

## Переменные окружения

Добавьте в файл `.env`:

```env
EMAIL_USER=clvx.fisy.ptue.acon@gmail.com
EMAIL_PASS=ваш_пароль_приложения_gmail
EMAIL_TO=kovaleva7855@gmail.com
```

## Тестирование

1. Запустите backend сервер
2. Перейдите на страницу анкеты потребителя
3. Заполните форму и отправьте
4. Проверьте почту kovaleva7855@gmail.com

## Структура письма

Письмо будет содержать:
- Информацию о пользователе
- Оценки удовлетворенности услугами
- Дополнительные комментарии
- Дату заполнения

## Безопасность

- Email отправителя скрыт от получателя
- Reply-To установлен на email пользователя для ответов
- Данные валидируются на сервере
